## CAD Designs-Prototype 

This folder contains all 3D‑designed holders required for the HSHL Prototyping project.
Each holder was modeled with correct tolerances, print‑ready geometry, and mounting features suitable for laser‑cut or 3D‑printed chassis integration.

## Included components 
## Motor holder 
-Designed to secure the DC motor firmly during motion.

-Includes side walls and a base slot for stability.

-Mounting holes added for attachment to the chassis.

-Clearance added to prevent vibration and friction.


## IR-sensor holder 
-Tolerances applied (0.3–0.5 mm) to ensure smooth insertion.

-Side guides prevent sensor wobbling.

-Front opening allows unobstructed IR transmission.


## UR-sensor holder 
-Dual‑eye cutouts for the ultrasonic transducers.

-PCB support rails included to hold the board in place.

-Mounting tabs added for chassis attachment.

-Fully closed geometry for 3D printing.


## Battery and Breadboard Holder 
-Compartment designed to hold the battery pack securely.

-Separate platform for the mini breadboard.

-Cable‑routing slots included for clean wiring.

-Reinforced walls for durability.


## Tools Used 
- Solid was used to design the parts.
  
- Bambu studio used to generate the simulation for the 3D printing
