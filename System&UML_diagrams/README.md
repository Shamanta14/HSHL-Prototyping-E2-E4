##  System & UML Diagrams

This folder contains the Unified Modeling Language (UML) and system design diagrams used to represent the architecture, behavior, and interactions of the autonomous robot system.
These diagrams help visualize how different subsystems (hardware and software) communicate and function together.


## Purpose 
-The diagrams in this folder serve to:

-Illustrate system structure and component relationships.

-Define subsystem interactions and data flow.

-Document requirements and design logic.

-Support debugging, integration, and future scalability.

-Each diagram provides a unique perspective on the robot’s design — from high‑level architecture to detailed operational flow.
